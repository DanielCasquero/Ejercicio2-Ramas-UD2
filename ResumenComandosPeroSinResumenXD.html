Los comandos que hemos dado en esta unidad (unidad de git y de uso de bash) se resumen en los necesarios para poder navegar en el CMD como pueden ser "cd" para moverse entre carpetas o "ls" para saber que hay dentro de una carpeta. Otros comandos nos pueden servir para crear directorios como "mkdir", o también para, dentro de esos directorios. crear archivos con sus respectivas extensiones, como puede ser "nano" o "touch". Una vez dentro de un directorio donde queramos iniciar un projecto de git tan solo escribimos "git init", eso crea una carpeta .git en el directorio, con eso ya podemos trabajar con git desde el git bash, ya dentro solo tenemos que, por ejemplo, abrir eclipse en esa carpeta, hacer lo que deseemos y posteriormente subirlo, para subir lo que hemos hecho en eclipse, tenemos que seguir un procedimiento, el cual es añadir los cambios y hacer un commmit, esto podemos hacerlo con "git add" y luego "git coommit -m "NombreDelComit" " o podemos resumirlo en "git commit -am "NombreDelComit" ", lo que añade el archivo antes de hacer el commmit, finalmente, tras hacer una commit podemos subirlo a un repositorio remoto mediante "git push URL main " (en caso de querer subirlo en la rama main) más generalmente podemos también podemos descargar el contenido de un repositorio remoto usando el "git pull" u otras cosas como hacer ramas con "git branch" o movernos entre ellas con "git checkout"
